# 🚀 Real-Time Chat Hub

A full-stack real-time chat application built using Node.js, Socket.io, and Vanilla JavaScript.

## 🛠 Features
- Real-time messaging using WebSockets.
- Dark mode modern UI.
- Auto-scrolling chat window.

## ⚙️ How to Run
1. Navigate to the `server/` folder.
2. Run `npm install`.
3. Start the server with `node index.js`.
4. Open `client/index.html` using a Live Server in VS Code.
